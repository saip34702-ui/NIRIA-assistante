NIRIA Assistant - An intelligent Python-based voice AI assistant with a web interface and local AI integration. The project demonstrates full-stack development combining voice processing, NLP, web development, and API integration.

🛠️ Tech Stack & Technologies
Backend: Python, Flask
AI/ML: Ollama, TinyLLama (Local LLM), OpenAI API integration
Voice Processing:
Speech Recognition (Google Speech API)
Text-to-Speech (pyttsx3)
Natural language translation (mtranslate)
Third-party APIs: OpenWeather API, PyWhatKit, Wikipedia, OpenAI GPT
Frontend: HTML, JavaScript
Packaging: PyInstaller (app.spec for executable distribution)
📋 Key Features Implemented
Voice Interaction

Hindi language voice input recognition with English translation
Female voice text-to-speech output with customizable speech rate
Intelligent Chat System

Local AI model integration using Ollama TinyLLama
Personality-based responses (NIRIA personality system)
Fallback to ChatGPT when needed
System Control & Integration

Open applications (Chrome, VS Code, etc.)
WhatsApp message scheduling with timezone support
Quick access to YouTube, WhatsApp Web
Information Retrieval

Real-time weather information via OpenWeather API
Wikipedia summaries for "who is" queries
Current time and date functionality
Web Interface

Flask-based web application with auto-browser launch
Chat interface for user interaction
RESTful API endpoint for message processing
